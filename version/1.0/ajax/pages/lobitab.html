LobiTab

LobiTab is jquery plugin for bootstrap tabs specially created for LobiAdmin. Using it you can add/remove/edit/sort tabs.

Basic Usage

LobiTab initialization is simple. Just create correct markup of bootstrap tabs and call plugin.

<div id="lobitab-demo">
    <!-- Nav tabs -->
    <ul class="nav nav-tabs" role="tablist">
        <li class="active"><a href="#home1" role="tab" data-toggle="tab">Home</a></li>
        <li><a href="#profile1" role="tab" data-toggle="tab">Profile</a></li>
        <li><a href="#messages1" role="tab" data-toggle="tab">Messages</a></li>
        <li><a href="#settings1" role="tab" data-toggle="tab">Settings</a></li>
    </ul>

    <!-- Tab panes -->
    <div class="tab-content">
        <div class="tab-pane active" id="home1">Home</div>
        <div class="tab-pane" id="profile1">Profile</div>
        <div class="tab-pane" id="messages1">Messages</div>
        <div class="tab-pane" id="settings1">Settings</div>
    </div>

</div>
<script>
    $(function(){
        $('#lobitab-demo').lobiTab();
    });
</script>

Basic example

Double click on tab title to edit it. Click anywhere outside or press enter to finish title editing. Click on mouse wheel to close tab. Drag tabs to reorder.

Custom options

Disable sorting, close button. Set newTabTitle to any text you like. Customize "Add new tab" button icon. You have enough control to get desired tabs.

$('#lobitab-demo-disable-close').lobiTab({
    closable: false,
    sortable: false,
    newTabTitle: 'Custom title',
    newTabContent: 'Content of new tab',
    addNewTabIcon: 'fa fa-plus'
});
