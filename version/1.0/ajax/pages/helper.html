Padding sizes

  • .padding-(3|5|8|10|12|15|20|25|30|35|40) creates padding on all sides of given size.

  • .padding-(vertical|horizontal|top|right|bottom|left)-(3|5|8|10|12|15|20|25|30|35|40) creates padding on specified side(s) of given size.

  • .padding-top-only, .padding-right-only, .padding-bottom-only, .padding-left-only removes padding on all sides except given one.

  • .padding-top-no, .padding-right-no, .padding-bottom-no, .padding-left-no removes padding on specified side.

  • .no-padding removes padding on all sides.

In order to create border, use any of the following classes

Class Name Description
.border-2
 Creates border-style: solid; 2px width border
.border-dotted-2
 Creates border-style: dotted; 2px width border
.border-dashed-2
 Creates border-style: dashed; 2px width border

  • .border-(solid|dashed|dotted)-(1|2|3) creates border on all sides of specific type

  • .border-(top|right|bottom|left)-(1|2|3) creates solid type border on specific side

  • .border-(top|right|bottom|left)-(dotted|dashed)-(1|2|3) creates border on specific side of specific style and size

  • .border-(top|right|bottom|left)-no disables border on specific side

  • .no-border disables border on every side

  • .border-transparent changes border color into transparent

Height helper classes

  • .h-(16|32|50|64|100|150|200|250|300|350) gives height to element

  • .h-full gives height: 100%

  • .min-h-(16|32|50|64|100|150|200|250|300|350) gives min-height to element

  • .max-h-(16|32|50|64|100|150|200|250|300|350) gives max-height to element

Width helper classes

  • .w-(16|32|50|64|100|150|200|250|300|350) gives width to element

  • .w-full gives width: 100%

  • .min-w-(16|32|50|64|100|150|200|250|300|350) gives min-width to element

  • .max-w-(16|32|50|64|100|150|200|250|300|350) gives max-width to element
